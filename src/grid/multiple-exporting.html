Grid supports client side exporting which allows you to export its data to Excel, Pdf and CSV formats.

In this demo, excelexport and pdfexport items are defined in toolbar. For these toolbar items, we have defined actions in toolbarClick event to export multiple Grids in same document using the excelExport, pdfExport methods.


Injecting Module

Grid features are segregated into individual feature-wise modules. To use exporting feature, we need to inject ExcelExportService and PdfExportService into the @NgModule.providers section.
