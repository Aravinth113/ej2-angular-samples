The Grid columns can be resized by clicking and dragging at the right corner of columns header. To enable column resize behavior, set allowResizing property as true. You can also prevent to resize the particular column by setting columns->allowResizing as false in columns definition. And, when double click at the right corner of column header, the respective column width will auto adjusted to its fit by autoFitColumns method.


In this demo, allowResizing feature have enabled through by setting the allowResizing property to true and Order ID column can be resized between the range of minWidth (120px) and maxWidth (300px). Also, column resizing has disabled in Shipped Date column.

Injecting Module

Grid component features are segregated into individual feature-wise modules. To use Resize feature, we need to inject ResizeService into the @NgModule.providers section.
