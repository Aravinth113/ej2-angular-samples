Reordering can be enabled by setting allowReordering property as true. Reordering can be done by drag and drop the particular column header from one index to another index within the Grid.

The location in which the column to be placed, will be indicated by two arrow symbol.

In this demo, you can reorder columns by drag and drop the column to the desired column.

Injecting Module

Grid component features are segregated into individual feature-wise modules. To use reordering feature, we need to inject ReorderService into the @NgModule.providers section.
