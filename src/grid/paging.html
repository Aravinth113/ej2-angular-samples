Paging allows you to display the contents of the Grid component in page segments. The number of items on a page is determined by the pageSettings->pageSize property. If no value is specified for the pageSettings->pageSize property, the Grid will display 12 items on a page. By default, paging is disabled. To enable paging, set allowPaging property to true.

In this demo, the Grid is rendered with its default page settings, also the pageSettings->pageCount is set to 5.


Injecting Module

Grid component features are segregated into individual feature-wise modules. To use paging feature, we need to inject PageService into the @NgModule.providers section.

More information on the paging feature configuration can be found in this documentation section.
