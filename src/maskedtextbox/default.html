The MaskedTextBox is a textbox extended component which allows the user to enter the valid input only based on the provided mask. It is used to get the input with specific formats like Phone number, Date, Time, IP, Social security number etc...

MaskedTextBox supports list of mask elements based on the standard MSDN mask elements.

In this demo, the mask was formed with standard mask elements to showcase the Phone number, Country code and so on.

More information about the MaskedTextBox instantiation can be found in this documentation section.
