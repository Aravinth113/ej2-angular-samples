In this sample, the DateTimePicker has been configured with the dd-MMM-yy hh:mm a date format. To change this current datetime format, go to the properties panel at the right side and select a date format from the dropdown options.

Datetime Formats sample illustrates the support of custom dateTime format in the DateTimePicker component by using the format property. You can also change the datetime format by selecting it from the format options in the properties panel.

More information on the date format configuration can be found in the documentation section.
