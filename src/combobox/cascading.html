The Cascading ComboBox is the series of ComboBox, where the value of one ComboBox depends on the another ComboBox value. This can be configured by making use of the change event of parent ComboBox. Within that change event handler, you should load the data to child ComboBox based on selected value of parent ComboBox.

In this sample, if a country is selected from countries ComboBox, then respective states will load in second ComboBox and the same has done between states and cities ComboBox.

More information on the Cascading feature configuration can be found in this documentation section.
