The ComboBox component allows the user to type a value, or choose an option from the list of predefined options.

The default sample illustrate the use of ComboBox that allowing end-users to select an item from options list and the selected item's value and text property values are displayed in the property panel. Also, this sample illustrates how to work the value property with two-way binding functionality in ComboBox.

More information on the ComboBox instantiation can be found in the documentation section.
