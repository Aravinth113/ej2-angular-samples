This sample demonstrates waterfall series for a company's various expenses and income.

In this example, you can see how to render and configure the Waterfall type series. Waterfall type charts are used for datas having five values in y-axis. You can use border, fill properities to customize the data appearance. dataLabel is used to represent individual data and its value.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use Waterfall feature, we need to inject WaterfallService into the @NgModule.providers section.
