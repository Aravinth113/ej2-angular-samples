This sample demonstrates Triangle Moving Average indicator in the chart.

In this example, you can see how to render and configure the TMA Indicator. TMA Indicator is used to represent the price movements in stock. You can use border, fill properities to customize the data appearance.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use TMA Indicator, we need to inject TmaIndicatorService into the @NgModule.providers section.

More information on the TMA Indicator can be found in this documentation section.
