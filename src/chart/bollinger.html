This sample demonstrates bollinger indicator in the chart.

In this example, you can see how to render and configure the BollingerBands. BollingerBands are used to represent the price movements in stock. You can use border, fill properities to customize the data appearance.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use BollingerBands, we need to inject BollingerBandsService into the @NgModule.providers section.

More information on the BollingerBands can be found in this documentation section.
