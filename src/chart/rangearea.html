This sample demonstrates range area series for temperature variation for a week.

In this example, you can see how to render and configure the range area type charts. You can use border, fill properities to customize the data appearance. dataLabel is used to represent individual data and its value.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use Candle feature, we need to inject RangeAreaService into the @NgModule.providers section.

More information on the range area series can be found in this documentation section.
