In this example, you can see how to render and configure the StepArea type charts. This series forms the step area progress, by connecting points through vertical and horizontal lines with area filled. You can use dashArray, width, fill properties to customize the area. marker and dataLabel are used to represent individual data and its value.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use step area series, we need to inject StepAreaService into the @NgModule.providers section.

More information on the step area series can be found in this documentation section.
