In this example, you can see how to render and configure the 100% stacking bar type charts. You can use border, fill properties to customize the vertical bar.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use 100% stacked bar series, we need to inject StackingBarService into the @NgModule.providers section.

More information on the 100% stacked bar series can be found in this documentation section.
