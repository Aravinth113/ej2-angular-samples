In this example, you can see how to render and configure the bubble type charts.A bubble chart is a type of chart that displays three dimensions of data by position and size of the bubble. Each points is drawn as a bubble, where bubble's size deponds on size You can use fill properties to customize the data appearance.

Tooltip is enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use bubble series feature, we need to inject BubbleService into the @NgModule.providers section.

More information on the bubbleseries can be found in this documentation section.
