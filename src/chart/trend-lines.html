By using property panel, you can change type of trend line by TrendLine Type, forward and backward forecasting of trendlines can be changed by Foreward Forecasting and Backward Forecast respectively. And also polynomial and period for a trenlines can be changed by Polynomial order and Period.

In this example, you can see how to render and configure the Trend Lines. Trend Lines is used to represent the price movements in stock. You can use border, fill properities to customize the data appearance.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use Trend Lines, we need to inject TrendlinesService into the @NgModule.providers section.
