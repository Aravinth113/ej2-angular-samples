In this example, you can see how to render and configure the range column type series. RangeColumn type charts are used for data having two values in y-axis. Those can be represented by high and low properities, which represent start and end point of columns in chart. You can use border, fill properities to customize the data appearance.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use RangeColumn feature, we need to inject RangeColumnService into the @NgModule.providers section.

More information on the column series can be found in this documentation section.
