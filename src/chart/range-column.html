This sample demonstrates range column series for temperature comparision of two countries for a week.

In this example, you can see how to render and configure the range column type series. RangeColumn type charts are used for datas having two values in y-axis. You can use border, fill properities to customize the data appearance. dataLabel is used to represent individual data and its value.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use RangeColumn feature, we need to inject RangeColumnService into the @NgModule.providers section.

More information on the column series can be found in this documentation section.
