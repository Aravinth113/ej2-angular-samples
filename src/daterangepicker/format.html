In this sample, the DateRangePicker has been configured with the dd/MMM/yy hh:mm a date format. To change this current datetime format, go to the properties panel at the right side and select a format from the dropdown options.

Format sample illustrates the support of custom date time format in the DateRangePicker component by using the format property. You can also change the date time format by selecting it from the format options in the properties panel.

More information on the date format configuration can be found in the documentation section.
