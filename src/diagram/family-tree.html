This sample visualizes the generations of a family.

In this example, you can see how to build a hierarchical tree from the external data source. You can use the dataSourceSettings property of the diagram to map the external data source with DiagramControl. You can use the layout property of the diagram to automatically arranges the nodes.

Injecting Module

Diagram component features are segregated into individual feature-wise modules. To generate diagrams from external data source, we need to Inject DataBinding module using Diagram.Inject(DataBinding) method. To automatically arrange the objects in a hierarchical structure, we need to Inject HierarchicalTree module using Diagram.Inject(HierarchicalTree) method.
