The Cascading DropDownList is the series of DropDownList, where the value of one DropDownList depends on the another DropDownList value. This can be configured by making use of the change event of parent DropDownList. Within that change event handler, you should load the data to child DropDownList based on selected value of parent DropDownList.

In this sample, if a country is selected from countries DropDownList, then respective states will load in second DropDownList and the same has done between states and cities DropDownList.

More information on the Cascading feature configuration can be found in this documentation section.
