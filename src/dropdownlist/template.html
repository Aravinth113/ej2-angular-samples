The DropDownList component provides a way to use a custom layout for its items and selected value which is shown in input using template feature.

In this sample, used below list of templates in DropDownList

  • ItemTemplate - To customize the list item's content.
  • ValueTemplate - To customize the value element content which holds the selected item's text.
  • HeaderTemplate - To customize the header element.
